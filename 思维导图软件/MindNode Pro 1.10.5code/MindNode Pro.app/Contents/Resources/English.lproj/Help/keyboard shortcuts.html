Keyboard Shortcuts



Node Management


<Return> Create new sibling node

<Tab> Create new child node

<Shift-Return> Create new main node

<Option-Return> Start editing title

<Backspace> Delete node



Navigation


<down arrow> Select next node

<up arrow> Select previous node

<left arrow> Select left node

<right arrow> Select right node

< Option-Command-> > Cycle main nodes (from left to right)

< Option-Command-< > Cycle main nodes (from right to left)



Positioning


Command-up arrow> Move node location up 1 pt

<Shift-Command-up arrow> Move node location up 10 pt

<Command-down arrow> Move node location down 1 pt

<Shift-Command-down arrow> Move node location down 10 pt

<Command-left arrow> Move node location left 1 ptove

<Shift-Command-left arrow> Move node location left 10 pt

<Command-right arrow> Move node location right 1 pt

<Shift-Command-right arrow> Move node location right 10 pt



Editing Title


be%<Option-Return> Forces a line break when editing the node title

<ESC> Abort title editing and restore node title



Folding


<Option-Command-left arrow> Fold Nodes

<Option-Command-right arrow> Unfold Nodes

<Option-Command-up arrow> Fold All Nodes

<Option-Command-down arrow> Unfold All Nodes



Inspector


<Option-Command-i> Show Inspector


Command key = Apple key

Option key = Alt key


MindNode also supports many commonly used Mac OS X keyboard shortcuts.

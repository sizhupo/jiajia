MindNode Tutorial


MindNode is a very simple and intuitive application for collecting, organizing and outlining ideas. In three very simple steps this tutorial shows how to interact with MindNode:


1. When starting MindNode, it will create a new document, create a new main node and put it into edit mode. This mode is indicated by a rounded rectangle. You can exit this mode by pressing Return or clicking on an empty part of the canvas.


2. You can create new nodes by hovering over a node and dragging the + sign. If you prefer you can just press the plus sign and a new node will be automatically created.


3. When you create a new node, you can edit it right away. You can enter the edit mode anytime by double clicking the node.

An introduction of other MindNode features is available in the User Guide.

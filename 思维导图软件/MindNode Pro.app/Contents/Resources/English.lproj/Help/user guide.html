User Guide



Creating Nodes


MindNode offers several ways to create new nodes. Probably the easiest way is to click and drag the node well (this is the plus sign that appears next to a node when you hover over it). There are also several keyboard shortcuts that help to quickly create nodes during brainstorming sessions.





Constrain Node Width


MindNode offers two different ways to constrain the node width:

  • Grow Width: In this mode the node size increases automatically until it reaches a predefined width. To enable this check the "Shrink Width To Fit" checkbox.
  • Fixed Width: Nodes have a fixed width and don’t grow or shrink automatically. To enable this uncheck the "Shrink Width To Fit" checkbox.


Hints

  • You can resize a node directly on the canvas by using the resize handle (the tiny squares that appear when a node is selected).
  • A simple double click on the resize handle will toggle width constraining.




Pro: Images and File Links


To insert an image or file link, drag and drop an image or file onto the canvas or onto an existing node. 


If you insert a file MindNode will create an abolsute link to the file. To open the file, press on the link icon or double click the file representation.




Pro: Reconnect Nodes


Sometimes it’s necessary to move a node from one branch to another. MindNode Pro makes this task extremely easy. Simply drag and drop a node onto another node.




MindNode Pro also allows to reconnect and reposition a node in one simple step: Drag a node onto an other node (a focus indicator will appear) and wait for the focus indicator to flash. The flash indicates that the two nodes are now connected and that you are able to move the node to a new location.


Hints

  • Pressing Space during hovering over a node will connect them immediately.
  • Pressing Option (Alt) during reconnecting, will apply the style of the super-node to its new sub-nodes.
  • If you accidently reconnected a node, you can press Esc to abort the operation.



Pro: Create Cross Connections


MindNode Pro allow to create a connection between any two nodes on the canvas.


The easiest way to create a connection between two nodes is to shift click on the originating node and while holding the mouse button, dragging a line to the destination node.




Pro: Fold Nodes


In MindNode Pro the node well is also used to fold and unfold nodes. When pressing Option (Alt) the plus sign transforms into the folding icon and a single click will fold or unfold the sub-nodes.





Pro: Create Links


To create a link, select a node and put it into the title edit mode (double click). Select the text which you want to underlay with a link. Open the inspector and select the text tab. Check the “Enable as hyperlink” check box and fill in the link into the text field (don’t forget to press Return after you entered the link).


To remove a link, select the relevant text and uncheck the “Enable as hyperlink” checkbox.


Hints

  • When enabling "Smart Link Detection" in the application preferences, the application will try to automatically detect links when entering text in the node title.

 



Pro: iPhone Sharing


MindNode Pro allows to use Wifi (Bonjour) to import and export documents from MindNode (touch). To share a document:

1. Access the sharing view in MindNode (touch): tap on the lower left icon on the main screen.

2. Open the iPhone Sharing Window in MindNode Pro: File > iPhone Sharing...

3. Select your mobile device from the list.

4. Approve the connection on your device.

5. Select a document from the Device or Local Open Documents list to transfer it to or from the device.


For WiFi sharing to work properly your Mac and your mobile device need to be connected to the same WiFi network.
